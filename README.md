LuntianHub is the secure, internal operating system for the Port Barton Marine Park Management Office. Unlike the public website, this platform is designed for Enforcement, Revenue Collection (LETS), and Cooperative Logistics (ICMS).

üöÄ Admin Access

This system uses a secure authentication protocol.

Authorized Personnel: Please contact the Chairman or Head of Enforcement for your Guardian ID and Passcode.

New Devices: Must be manually registered by an Admin in the "Manage Guardians" tab.

Security Note: This application is currently running in Client-Side Mode for the Alpha phase. All data is stored locally on the device for privacy compliance until the cloud migration (Phase 3).
